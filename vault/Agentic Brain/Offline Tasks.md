# Offline Tasks

Recorded workflows TONY replays without internet.

## How it works

1. **Online**: TONY completes a task using tools (2+ steps auto-recorded)
2. **Stored**: Steps saved to `data/tasks.json` with triggers
3. **Offline**: Say `repeat <name>` — replays local tools only

## Triggers

- English: `repeat X`, `run task X`, `do X again`
- Urdu/Roman: `X phir se`, `X dobara`

## Offline tools

memory, graphify, obsidian, read_file, list_directory, shell (allowlist)
