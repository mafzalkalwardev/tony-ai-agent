# Code Workflows

Polyglot development workflows — scaffold, test, deploy, automate.

## Workflow pattern
1. **Understand** — codegraph_context or openwiki_search
2. **Plan** — workflow_run for multi-step; goal_run until success criteria pass
3. **Implement** — write_file + shell; match repo conventions
4. **Verify** — npm test / pytest; error_learn on failures
5. **Ship** — git commit only when user asks; gh pr create when requested

## Languages
JavaScript/TypeScript, Python, Rust, Go, Java, C#, PHP, Ruby, Swift, Kotlin, SQL, shell.

## Automation
- **desktop_automate** — pyautogui click/type/hotkey/screenshot on Windows
- **playwright** MCP — browser automation
- **scraper_media** — research before building

## Jan.ai local LLM
When cloud rate-limited, Jan at localhost:1337 provides offline OpenAI-compatible inference.
