# Friday executive brief skill

Inspired by executive-assistant agents (Friday pattern).

## When to use

- End-of-day or weekly summaries
- Priority triage across projects
- "What should I focus on?" questions

## Workflow

1. Pull SignalMint workspace stats if relevant
2. Search memory for open threads and user preferences
3. Structure output as:
   - **Wins** — completed since last session
   - **Focus** — top 3 priorities
   - **Watch** — risks or blockers
   - **Quick wins** — under 15 minutes each

## Tone

Executive brief: scannable bullets, numbers where available, no fluff.
