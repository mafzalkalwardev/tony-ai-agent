# SignalMint operations skill

Use for SMS marketing, campaigns, compliance, and workspace admin.

## Tools

- `signalmint_health` — API status
- `signalmint_workspace` — usage and lines
- `signalmint_conversations` — inbox list

## Prerequisites

Set in `.env`:
- `SIGNALMINT_API_URL`
- `SIGNALMINT_EMAIL`
- `SIGNALMINT_PASSWORD`

## Safety

- Sandbox mode by default in dev
- Confirm before sending live SMS or launching campaigns
