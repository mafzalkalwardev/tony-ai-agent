# Research skill

Use when gathering information from memory, knowledge index, web, or GitHub.

## Workflow

1. `memory_search` for prior research
2. `knowledge_search` for OSS agent patterns
3. `web_fetch` for public docs (when URL known)
4. `github_search` when token configured
5. Synthesize with citations; store key facts in memory

## Sources priority

1. Local codebase and memory
2. Curated agents-index.json
3. Public HTTP
4. GitHub API
